Robotics

Autonomous, programmable Arduino robots.

In my spare time I have worked on developing robotics projects using Arduinos. I have created two prototype robots, with autonomous functionality and the ability to be programed. These I created from scratch, building the structure, completing all the wiring, and developing all hardware code and associated Java applications. My aim is to develop an autonomous rover able to record weather data. This I hope to combine with skills learnt from developing DigiSign, to allow the robot to be controlled, and allow the user to see any data, in real-time through a web application.
